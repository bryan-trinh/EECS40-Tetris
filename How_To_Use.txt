Keyboard controls:
*Note that the control will perform 1 unit of the move
**If the Key is held, then the action performed will correspond
	to however long the key is held

Key				Control
____________________________	__________________
Left Directional Arrow Key 	- Move Left
Right Directional Arrow Key 	- Move Right
Down Directional Arrow Key 	- Move Down
Q 				- Rotate Left
E 				- Rotate Right
S				- Drop Down
F				- Hold/Swap Piece

- 1000 points per row cleared
- Can clear multiple rows (still 1000 points per row)
- Holding pieces will only put the current piece into a "bank"
	where the user can swap that piece in the bank to use
	at a later time. The first time using this piece allows
	the user to put it into the bank and take in the next
	piece. The user can decide to swap the selected piece
	out or keep on playing per usual.

Deliverables:
- Can use touchscreen buttons as control (encoded by onClick)
- Can use keyboard as control
